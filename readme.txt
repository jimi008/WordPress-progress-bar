== Description ==
An automatic web page progress bar. WordPress progress bar automatically add beautiful progress and activity indicators for page loads and ajax navigation. It is free and open source.

== Installation ==
In most cases you can install automatically from WordPress. You will be taken to the plugin\'s settings page, where you can customize and complete your installation.

However, if you install this manually, follow these steps:

Upload the entire wp-progress-bar folder to the /wp-content/plugins/ directory.
Activate the plugin through the \'Plugins\' menu in WordPress.
